## To Run Locally

1. **Clone the repository:**

   ```
   git clone https://github.com/nasib15/scic-server.git
   cd scic-server
   ```

2. **Install the dependencies:**

   ```
   npm install
   ```

3. **Start the development server:**

   ```
   npm run dev
   ```

### Build for Production:

```
npm run build
```
